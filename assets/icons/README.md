# App Icons

Place a custom icon here to update the app icon in the UI and Windows installer.

## Supported filenames

Add **one** of the following filenames:

- `app_icon.ico` (preferred for Windows packaging)
- `app_icon.png`
- `app_icon.jpg`
- `app_icon.jpeg`

If you supply a PNG/JPEG, the build script will auto-generate an `.ico` file for
the Windows executable.
